# Expense Tracker (FastAPI)

A full-stack **Expense Tracker web application** built using **FastAPI, SQLite, and HTML/CSS**.  
This project allows users to manage their daily expenses, view summaries, and receive expense reports via email.

## Features

- User Authentication (Signup & Login)
- Add and manage expenses
- Dashboard overview with total spending
- Budget tracking
- Email expense report
- SQLite database integration
- REST API using FastAPI

## Tech Stack

Backend:
- FastAPI
- Python
- SQLAlchemy
- SQLite

Frontend:
- HTML
- CSS
- JavaScript
